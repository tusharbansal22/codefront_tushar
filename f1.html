WHY LEARN HTML?

Knowing how to code in any language is an invaluable skill. Despite how complex websites and apps may be, they are still built with the fundamental blocks of the web: HTML & CSS. These languages are renowned for being beginner-friendly and have a great community that surrounds them. Learning how to code allows you to bring out your personality on your own website, enables you to stand out from the crowd, contribute to projects better, helps you understand and explore other languages and provides you with an opportunity to transition into a different career.

HTML

HTML is the standard markup language for Web pages.
With HTML you can create your own Website.
HTML is easy to learn - You will enjoy it!


HTML code starts with html starting and closing tag

<html> "your code" </html>

html tag is followed by head tag in which we can specify title

<html> <head> <title> "Your title" </title> </head> </html>

After that we specify body of the website

<html> <head> <title> 'hello world' </title> </head> <body> <p>hello</p> </body> </html>

And that's it!
This is basic syntax of html.

CSS

CSS is basically used for styling html. All the basic structure of websites lies in Html itself but css makes it presentable.

For using css we make individual groups of elements called classes. We can call any class in CSS using a "." . Apart from that we can also change styling for a particular type of element at once.
Firstly, lets understand some attributes of CSS

FONT FAMILY

It changes the font of the text

font-family: arial;

WEIGHT

It changes boldness of text

font-weight:bold;

COLOR

It changes the colour of the text

color:white;

PADDING

It changes the internal spacing in a box element

padding:10px 15px;

MARGIN

It changes external spacing in a box element

margin:15px 20px;
